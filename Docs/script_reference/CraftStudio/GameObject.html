GameObject.BroadcastMessage (go, functionName, data, gameObject)
Tries to call a method with the specified name on all the scripted behaviors attached to the gameObject or any of its descendants. The data argument can be nil or a table you want the method to receive as its first (and only) argument. If none of the scripted behaviors attached to the game object or its children have a method matching the specified name, nothing happens. Uses GameObject:SendMessage() on the gameObject and all children of its children.

Parameters:

  • go:
  • functionName: (string) The method name.
  • data: [optional] (table) The data to pass along the method call.
  • gameObject: (GameObject) The gameObject

GameObject.DestroyComponent (go, input, strict, gameObject)
Destroy a component from the gameObject. Argument 'input' can be : the component type (string) (the function will destroy the first component of that type), the component object or instance (ScriptedBehavior, Model, Map or Camera), or a script name or asset (string or Script) (the function will destroy the ScriptedBehavior that uses this Script),

Parameters:

  • go:
  • input: (mixed) See function description.
  • strict: [optional default=false] (boolean) If true, returns an error when the function can't find the component to destroy.
  • gameObject: (GameObject) The gameObject

Return value:

    (boolean) True if the component has been succesfully destroyed, false otherwise.
GameObject.DestroyScriptedBehavior (go, scriptNameOrAsset, gameObject)
Destroy a ScriptedBehavior form the gameObject. If argument 'scriptNameOrAsset' is set with a script name or a script asset, the function will try to destroy the ScriptedBehavior that use this script. If the argument is not set, it will destroy the first ScriptedBehavior on this gameObejct

Parameters:

  • go:
  • scriptNameOrAsset: [optional] (string or Script) The script name or asset.
  • gameObject: (GameObject) The gameObject

Return value:

    (boolean) True if the component has been succesfully destroyed, false otherwise.
