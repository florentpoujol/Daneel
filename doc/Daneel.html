Daneel.Debug.CheckArgType (argument, argumentName, expectedArgumentTypes, p_errorHead, p_errorEnd)
Check the provided argument's type against the provided type(s) and display error if they don't match.

Parameters:

  • argument: (mixed) The argument to check.
  • argumentName: (string) The argument name.
  • expectedArgumentTypes: (string or table) The expected argument type(s).
  • p_errorHead: [optional] (string) The beginning of the error message.
  • p_errorEnd: [optional] (string) The end of the error message.
Daneel.Debug.CheckAssetType (assetType)
Check the value of 'assetType', correct its case or convert it to string and throw error if it is not one of the valid asset types or objects.

Parameters:

  • assetType: (string, Script, Model, ModelAnimation, Map, TileSet, Scene or Sound) The asset type as a string or the asset object.

Return value:

    (string) The correct asset type.
Daneel.Debug.CheckComponentType (componentType)

Parameters:

  • componentType: (string, ScriptedBehavior, ModelRenderer, MapRenderer, Camera or Transform) The component type as a string or the asset object.

Return value:

    (string) The correct component type.
Daneel.Debug.CheckOptionalArgType (argument, argumentName, expectedArgumentTypes, p_errorHead, p_errorEnd)
If the provided argument is not nil, check the provided argument's type against the provided type(s) and display error if they don't match.

Parameters:

  • argument: (mixed) The argument to check.
  • argumentName: (string) The argument name.
  • expectedArgumentTypes: (string) The expected argument type(s).
  • p_errorHead: [optional] (string) The beginning of the error message.
  • p_errorEnd: [optional] (string) The end of the error message.
Daneel.Debug.GetType (object, returnLuaTypeOnly)
Return the Lua or CraftStudio type of the provided argument. "CraftStudio types" includes : GameObject, ModelRenderer, MapRenderer, Camera, Transform, Script, Model, ModelAnimation, Map, TileSet, Scene, Sound, Document, Ray, RaycastHit, Vector3, Plane, Quaternion

Parameters:

  • object: (mixed) The argument to get the type of.
  • returnLuaTypeOnly: [optional default=false] (boolean) Tell whether to return only Lua's built-in types (string, number, boolean, table, function, userdata or thread).

Return value:

    (string) The type.

Daneel.Events.Fire (eventName, ...)
Fire the provided event transmitting along all subsequent parameters to 'eventName' if some exists. All functions that listen to this event will be called and receive all parameters.

Parameters:

  • eventName: (string) The event name.
  • ...: [optional] A list of parameters to pass along.
Daneel.Events.Listen (eventName, p_function, functionName, broadcast)
Make the provided function listen to the provided event. The function will be called whenever the provided event will be fired.

Parameters:

  • eventName: (string) The event name.
  • p_function: (function, string or GameObject) The function (not the function name) or the gameObject name or instance.
  • functionName: [optional default="On[eventName]"] (string) If 'p_function' is a gameObject name or instance, the name of the function to send the message to.
  • broadcast: [optional default=false] (boolean) If 'p_function' is a gameObject name or instance, tell whether to broadcast the message to all the gameObject's childrens (if true).
Daneel.Events.StopListen (eventName, functionOrGameObject)
Make the provided function or gameObject to stop listen to the provided event.

Parameters:

  • eventName: (string) The event name.
  • functionOrGameObject: (function, string or GameObject) The function, or the gameObject name or instance.
